[2020.10.14]安装步骤

###### 初始安装
    pip3 install jsbsim

###### 下载源码并解压
    https://github.com/JSBSim-Team/jsbsim/releases/tag/Linux
    
######  准备编译 jsbsim
    cd jsbsim  
    mkdir build 
    cd build
    cmake .. 

###### 编译 jsbsim
    make 

###### 其它
    更改 /Mini0jsbsim/simulation.py 里的 ROOT_DIR 为 jsbsim 解压后的文件夹
    将模型 <Rascal110_JSBSim> 拷贝到 /aircraft 文件夹下
